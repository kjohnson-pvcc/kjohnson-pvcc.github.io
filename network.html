OPERATING SYSTEMS TERMINOLOGY
TERM DEFINITIONS
Network Classifications
    LAN (Local Area Network) - a collections of computers usually in a single building or building complex
    MAN (Metropolitan Area Network) - a local community network of intermediate size
    WAN (Wide Area Network) - links networks in neighboring cities or even worldwide
Network Models
    Bus Network - machines within this network communicate with each other over a common bus, buses have short links
    Star Network - machines within this network comminucate with each other over a itermediary central machine, stars have longer links and are shaped like a star
Protocols
    CSMA/CD Ethernet - controls the trasmittals of messages in a bus network
    CSMA/CA Wifi - wireless networks try to avoid collisons instead of trying to detect them
Combining Networks
    Repeater - passes signals back and forth between the two original buses without considering the meaning of the signals
    Bridge - connects two buses but doesn't pass all messages across the connection, it only forwards a message when a other computer is on the receiving end
    Switch - essentially a bridge, except it can connect several buses
    Router - the conection between networks to form internet is handled by this device, which it's purpose is used to forward messages
    Gateway - when a network is linked to the internet, because it's a passageway to the outside world and the network
Client/Server Model
    P2P: peer-to-peer - involves the server provding a service to numberous clients, where that process is to provide and receive service from each other
    Distributed System - software units that is capable of executing it's processes on different computers
    Cloud Computing - shared computers on a network can be allocated for use by clients as needed
Internet Adressing
    Tier 1 - ISPs with high-speed, high-capacity international WANs
    Tier 2 - ISPs that are more regional and have less capabilities, tend be operated by companies in communications
    Tier 3 - ISPs supplying internet to individual homes or businesses, tend to be cable or telephone companies
IP Address
    ICANN - nonprofit corporation that was established to coordinate the operation of the Internet
    DNS - name servers that are used as a directory for the Internet
    TLD: Top-Level Domain - the classification of a domain
    Country-Code TLDs - two letter TLDs for each specific country
Internet Protocols
    FTP (File Transfer Protocol) - application used to access another computer from far distances
    HTTP (Hyper Text Transfer Protocol) - used to encode and transport information between a client and server
    SMTP (Simple Mail Transfer Protocol) - allows two computers to on the network to interact when sending email messgaes between hosts
World Wide Web
    URL (Uniform Resource Locator) - a unique address given to documents to allow them to be located and retrieved
    Client-Side Activities - activites performed by a client (ex. a browser filling out an order form)
    Server-Side Activities - activites performed by the server (ex. the server sends the customer an email)
Internet Four-Layer Software
    Application Layer - consists of software units such as clients and servers that carry out their tasks using Internet communication
    Transport Layer - accepts messages from the applciation layer and makes sure they a properly formatted to send over the Internet
    Network Layer - decides which direction a packet should be sent to at each step
    Link Layer - sends the packet to either back to the network layer or forwards the packet on it's way to the final destination
Internet is:
    Packet-Switching Network - messages are broken down into small groups of data known as packets
    Fault Tolerant - a systems ability to continue to operate when there may be failure of one of the components
    Redundant - allows more than one way to route data between points on the internet
Cybersecurity
    Malware - malicious software used to attack a computer
    Virus - software that infects a computer by inserting itself into programs already on the machine
    Worm - a autonomous program that transfers itself through a network, taking uo residence on a computer, and forwarding copies of itself to other computers
    Trojan Horse - enters a computer by disguising itself as a desireable program that is willing imported by the user of the computer
    Spyware - another malicious software that collects information about activites performed on the computer and reports back to the instigator of the attack
    Phishing - a technique of trying to obtain information by just asking for it
    DDoS Attack - overloading many host computers with a large amount of messages in a short period of time
Protection
    Firewall - used to filter traffic passing through a network point
    Spam Filter - a firewall used to block unwanted emails
    Proxy Server - a software unit whose goal is to shield the client from adverse actions from the server
    Antivirus Software - software used to remove known viruse and other infections from your computer
    Crypography - the study of sendy and receiving messages securely in the presence of adversaries
    Public Key Encyption - the encryption and decryption algorithms are known and one of the keys are known, but the encoded messages stays secure
