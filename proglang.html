TERM DEFINITIONS
Machine Language a sequence of instructions that are encoded using numeric digits
Debugging locating and correcting errors in a program
Assembler a program that converts mnemonic expressions into machine language instructions
Assembly Language a mnemonic system to represnt programs
Translator a program that translates high-level primitives into machine language programs
Compiler used to compile several machine instructions into short sequences to simulate activity requested by a single high level primitive
Interpreter executes the instructions as they are translated instead of producing a machince-language copy for future use
Cross-Platform software software that is independent of an operating system's design as well of the machine's hardware design
Natural Language languages such as English, German, and Latin
Formal Language languages such as programming languages
Machine Independence machines that do not rely on the characteristics of a particular machine
High-Level Programming Language has a significant abstraction from the details of computer operation
Object-Oriented Programming a software system that is viewed as a collection of units called objects, each that are capable of performing the actions that are immediately related to itself as well as requesting the actions of other objects
Scripting Language a collection of languages used to program, these languages are typically used to perform administrative tasks rather than develop complex programs
