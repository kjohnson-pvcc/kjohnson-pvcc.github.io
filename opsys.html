OPERATING SYSTEMS TERMINOLOGY
TERM DEFINITION
Operating system the software that controls the overall operation of a computer.
Job Queue a storage organization in which jobs are ordered in first in, first out
FIFO stands for first in, first out, pronounced "FI-foe"
Interactive processing operating systems that allows the program being used to communicate with the user through remote terminals
Time-sharing operating systems that provide service to multiple users at the same time
Multiprogramming dividing time between multiple jobs so that only one can be executed at a time. Jobs are shuffled back and forth to create the illusion of several jobs being executed at once
Application software programs used to perform tasks particular to the machine's utilization
System software performs taks that are common to a computer's system in general
Utility software programs used to perform activities that are fundamental to a computer installation, but not the operating system
Graphical user interface (GUI) files and programs are represented as pictures on the display as icons, pronounced "GOO-ee"
Shell use textual messages to communicate with users via the keyboard and monitor screen, this is an older user interface
Kernel the internal part of an operating system which contains the software components that perform very basic functions required by the computer installation
Device driver software that communicates with controllers or devices to carry out operations on the peripheral devices that are connected to the machine
File manager to coordinate the use of the mass storage facilities within the computer
Memory manager has the task of maintaing the machine's use of main memory
Virtual Memory "fictional" memory space created by using data pages to store more memory
Boot Strapping transfers the operating system from mass storage to main memory whenever it is first turned on
ROM read-only memory; a computer's memory that can only be read and not altered
Deadlock when twor or more processes are blocked from progressing because they are waiting for a resource allocated to another
Spooling to hold data to output at a later more convenient time
Super user (or administrator) has highly privileged access to the operating system. Can alter settings within the operatimg system, adjust privileges of other users, and modify software packages.
Multi-factor authentication having the user require more than one authentification factor to gain access, this could include a combination of a password, security questions, special access code, and/or biometric sensor
